# S3-T8: Convergence Pattern Classification

| Pattern                |   Count | Description                                          |
|:-----------------------|--------:|:-----------------------------------------------------|
| Get it right first try |      18 | Iter 1 improves, iters 2-3 may regress               |
| Progressive refinement |       2 | Valid & improves vs previous on ALL 3 iterations     |
| Late bloomer           |      12 | Peaks at Iter 2 or 3, without continuous improvement |
| Stuck at seed          |      36 | No iteration beats the seed                          |
